# Customer Segmentation using RFM Analysis (Power BI)

## Overview
This project analyzes customer behavior using **RFM (Recency, Frequency, Monetary)** analysis to segment customers into meaningful groups and identify value, risk, and engagement patterns.

The dashboard is designed in two levels:
- **Page 1: Overview** – high-level distribution and value summary
- **Page 2: Deep Dive** – detailed behavioral analysis and insights

## Tools Used
- Power BI
- DAX
- RFM Analysis
- Excel / CSV Dataset

---

## Key Insights
- At-Risk Spenders generate high revenue but show poor recency → win-back opportunity
- Best Customers dominate frequency and value → retention priority
- New Customers show low frequency → activation required
- At-Risk Loyals show declining engagement → churn risk
